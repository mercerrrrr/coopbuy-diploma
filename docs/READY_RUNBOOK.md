## 10-минутный сценарий

### 1. ADMIN — Дашборд (~2 мин)

1. Войти как `admin@local.test / Admin123!`
2. Открыть **`/admin/dashboard`** — видны 3 карточки закупок с KPI
3. Перейти **«К списку закупок»** — полный список с фильтрацией
4. Открыть **«Закупка №3 — АгроОпт (завершена)»** → детали закупки
5. Нажать **«Отчёт»** → `/admin/procurements/[id]/report`
   (таблица участников, топ категорий/товаров)
6. Нажать **«Оплаты XLSX»** — файл скачивается с реестром оплат

**Ожидается:** карточки отображаются, отчёт генерируется, XLSX скачивается.

---

### 2. RESIDENT — Просмотр заявок и уведомлений (~2 мин)

1. Войти как `user1@local.test / User123!`
2. Открыть **`/my/orders`** — 3 заявки (2 OPEN-закупки + 1 CLOSED)
3. Кликнуть на заявку → видны QR-код и кнопка **«Скачать квитанцию (PDF)»**
4. Открыть **`/my/notifications`** — 5 уведомлений (все непрочитанные)
5. Нажать **«Отметить все прочитанными»** — badge в шапке исчезает

**Ожидается:** заявки отображаются, QR-код виден, уведомления читаются.

---

### 3. OPERATOR — Выдача заказа (~2 мин)

1. Войти как `operator1@local.test / Operator123!`
2. Открыть **`/admin/procurements`** — видна закупка №1 (с привязкой ПВЗ Центральный)
3. Открыть деталь закупки → секция **«Выдача»** — сессия уже активна
4. В поле **orderId** вставить ID заказа user2 (PAY_ON_PICKUP) → нажать **«Выдать»**

**Ожидается:** checkin создан, заказ переходит в статус «Выдан».

---

### 4. ADMIN — Приёмка и импорт (~2 мин)

1. Войти как `admin@local.test / Admin123!`
2. Открыть **«Закупка №3 — АгроОпт (завершена)»** → **«Акт приёмки»**
   — видны 2 строки (гречка с расхождением, молоко без расхождения)
3. Открыть **`/admin/suppliers`** → зайти в «АгроОпт-Юг»
4. Перейти во вкладку **«Импорт»** — загрузить CSV-файл с прайсом
5. Открыть **`/admin/dictionaries`** — управление категориями и единицами

**Ожидается:** акт приёмки отображает расхождения, импорт CSV работает.

---

### 5. ADMIN — Экспорт отчётов (~2 мин)

1. Открыть **«Отчёт»** по завершённой закупке
2. Скачать **report.pdf** — PDF с KPI и таблицей заказов
3. Скачать **report.xlsx** — 4 листа: Summary, Orders, TopCategories, TopProducts
4. Вернуться на страницу закупки → **«Экспорт XLSX»** — список товаров

**Ожидается:** все форматы скачиваются, данные заполнены.

```bash

npm run dev

# Запустить тесты
```
